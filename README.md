# Cloud PACS Integration Platform

A healthcare interoperability project built in Python.

## What this does
- Parses HL7 v2 ADT messages
- Transforms to FHIR R4 Patient resources  
- Posts to a live FHIR server via REST API

## Technologies
Python · HL7 v2 · FHIR R4 · REST API · Git
